## Ксюша
- Досвідчений UI/UX дизайнер
- Основний інструмент — Photoshop
- Illustrator для конвертації SVG → PSD
- Стиль універсальний, залежить від проєкту
- Спілкується українською та англійською

## Активний проєкт
- **Casino/gaming banners** — широкоформатні баннери 1050×350px
- Нова тематика: **казино інтерфейс на смартфоні, ставки на смартфоні**
- Стиль: hyper-realistic 3D CGI renders, glossy cartoon-realism
- **GGBet brand style** — помаранчево-чорна колірна гама
- Композиція: центральний смартфон з casino interface + floating casino objects
- Колірна гама: `#FF6B00`/`#FF8C00` (GGBet orange) + `#0d0d0d`/`#1a0a00` (чорний фон)

## Патерни роботи
- Надсилає референси через forward з іншого чату
- Просить варіації: "такого ж розміру і у такому ж стилі"
- Реалістичні/3D assets → Gemini, UI компоненти → HTML
- Приймає роботи через /learn — база в style_knowledge.md
- Точні правки: "розверни на 45 градусів", "заміни центральний об'єкт"
- Відхиляє неточності: "не ясно що на тему казино", "ніколи радянські символи"
- Заборона п'ятикутних зірок та радянської символіки
