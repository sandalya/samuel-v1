## Ксюша
- Досвідчений UI/UX дизайнер
- Основний інструмент — Photoshop
- Illustrator для конвертації SVG → PSD
- Стиль універсальний, залежить від проєкту
- Спілкується українською та англійською

## Активний проєкт
- **Sports betting banners** — широкоформатні баннери 1050×350px
- Перехід від gaming/casino до спортивних ставок
- Стиль: photorealistic 3D CGI renders, темна спортивна атмосфера
- Теми: футбол + клуб Celtic + тренери + спортивні ставки
- Композиція: триплановість — герої на передньому плані, команда в тіні на задньому
- Колірна гама: deep dark backgrounds, dramatic lighting, cinematic depth

## Патерни роботи
- Надсилає референси через forward з іншого чату
- Просить варіації за референсом: "такого ж розміру і у такому ж стилі"
- Реалістичні/3D assets → Gemini, UI компоненти → HTML
- Приймає роботи через /learn — база в style_knowledge.md
- Уточнює деталі: "заміни лише обличчя", "додай тренера"
- Надсилає фото для копіювання зовнішності
